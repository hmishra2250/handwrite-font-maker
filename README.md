# handwrite-font-maker

Convert a handwriting specimen sheet into installable font files.

The pipeline in this repository is designed for the boxed worksheet format in the two sample images you shared. It handles:

- grid detection
- guide-line cleanup
- bitmap smoothing
- vectorization with Bezier curves
- font generation through FontForge

## Status

The repository scaffold is in place. The build pipeline and usage docs are added in follow-up commits.
